# psp_2048

2048 written in Python 3 with GUI

*Not tested on Windows

### Prerequisites

```
System with Python 3.x onboard
Installed packages: py-getch, termcolor, tkinter
```

### Installing

```
pip3 install py-getch
pip3 install termcolor
pip3 install tkinter
```

### Quick start guide

```
cd "PROJECT_LOCATION"
python3 main.py
```
